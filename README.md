# 나라배움터 교육과정 지원 챗봇

국민건강보험공단 나라배움터(e-learning.nhi.go.kr) 집합교육 과정 수강생을 위한 카카오 챗봇 및 개인 알림 도구입니다.

---

## 목적

집합교육 기간 동안 교육생이 **일정 확인**과 **공지사항 모니터링**을 편리하게 할 수 있도록 지원합니다.

- 카카오톡에서 날짜를 입력하면 해당 일자의 교육 일정을 즉시 확인
- 나라배움터에 새 공지가 올라오면 카카오 알림톡으로 자동 수신

---

## 프로젝트 구조

```
claude_projects/
├── kakao_bot/          # 공개 챗봇 (Render 배포)
└── local_notifier/     # 개인용 로컬 알림 도구
```

### kakao_bot — 공개 챗봇

카카오 i 오픈빌더와 연동된 웹훅 서버입니다. Render 무료 플랜으로 운영합니다.

**사용 가능한 기능**

| 입력 예시 | 결과 |
|-----------|------|
| `오늘 일정` | 오늘 날짜 교육 일정 출력 |
| `내일 일정` | 내일 날짜 교육 일정 출력 |
| `6월 15일 일정` | 특정 날짜 교육 일정 출력 |

**기술 스택**
- Python / FastAPI / uvicorn
- Render 무료 플랜 + UptimeRobot (슬립 방지)

**로컬 실행**
```bash
cd kakao_bot
pip install -r requirements.txt
uvicorn main:app --reload
```

### local_notifier — 개인용 알림 도구

나라배움터에 로그인해서 과정 공지사항을 주기적으로 확인하고, 새 공지 발생 시 카카오 알림톡을 발송합니다. 로컬(한국 IP)에서만 실행합니다.

**사용 가능한 기능**

- 15분 간격으로 나라배움터 과정 공지사항 자동 확인
- 신규 공지 감지 시 카카오 알림톡 자동 발송

**설치 및 실행**
```bash
cd local_notifier
pip install -r requirements.txt
cp .env.example .env   # .env에 실제 값 입력
python monitor.py
```

**필요한 환경변수 (.env)**
```
NARAEBAEUM_ID=나라배움터_아이디
NARAEBAEUM_PW=나라배움터_비밀번호
NARAEBAEUM_COURSE_ID=과정_세션_ID
```

> **참고** `NARAEBAEUM_COURSE_ID`는 나의 강의실에서 과정 클릭 시 전달되는 `sbjectSessId` 값입니다.

신규 공지 발생 시 macOS 시스템 알림(화면 오른쪽 상단 팝업)으로 알림을 받습니다.

## 앞으로의 개발 방향

개발 기능은 두 가지 방향으로 분리해서 진행합니다.

### 공개 챗봇에 추가할 기능
- [ ] 교육운영안내 책자 기반 자주 묻는 질문(FAQ) 응답
- [ ] 강의실 위치, 생활관 안내 등 교육 인프라 정보 제공
- [ ] 주차별 학습 현황 안내

### 개인용 로컬 도구에 추가할 기능
- [ ] 결석/결강원 신청 일정 알림
- [ ] 과제 제출 마감 알림
- [ ] 알림 조건 커스터마이징 (특정 키워드 포함 공지만 알림)

## 주의사항

- `local_notifier`는 나라배움터 로그인이 필요한 정보를 처리합니다. 개인 용도로만 사용하세요.
- 해외 IP(Render 등)에서는 나라배움터 접속이 차단되므로 반드시 로컬(한국 IP)에서 실행해야 합니다.
